Problem 1: Prove the Sum of First n Natural Numbers

Prove that the sum of the first n natural numbers, i.e., 0 + 1 + 2 + ... + n, equals n*(n+1)/2.

Solution:

        Base Case:
        P(0) = 0 = 0*(0+1)/2.
  
        Induction Hypothesis:
        Assume P(n) holds, i.e., sum 0 + 1 + ... + n = n*(n+1)/2.
  
        Induction Step:
        Show that P(n+1) holds, i.e., sum 0 + 1 + ... + n + (n+1) = (n+1)*(n+2)/2.
        Using the induction hypothesis:
        n*(n+1)/2 + (n+1) = (n+1)*(n+2)/2.

Problem 2: Prove Sum of Squares Formula

Prove that the sum of squares of the first n natural numbers, i.e., 0^2 + 1^2 + 2^2 + ... + n^2, equals n*(n+1)*(2n+1)/6.

Solution:

        Base Case:
        P(0) = 0^2 = 0*(0+1)*(2*0+1)/6.
  
        Induction Hypothesis:
        Assume P(n) holds, i.e., sum 0^2 + 1^2 + 2^2 + ... + n^2 = n*(n+1)*(2n+1)/6.
  
        Induction Step:
        Show that P(n+1) holds, i.e., sum 0^2 + 1^2 + 2^2 + ... + n^2 + (n+1)^2 = (n+1)*(n+2)*(2n+3)/6.
        Using the induction hypothesis:
        n*(n+1)*(2n+1)/6 + (n+1)^2 = (n+1)*(n+2)*(2n+3)/6.
