Lesson 8: Lists, Part 2

Introduction to Lists

Lists in functional programming are sequences of elements of the same type, such as [2, 18, 9, 10]. The list type is denoted as [Int], representing a list of integers.

Definition

A list is either empty ([]) or consists of a head (single element) and a tail (another list). These are connected by the :: operator.

Example: [2, 18, 9, 10] is equivalent to 2::(18::(9::(10::[]))).

Functions

Function Signature Example
head [a] -> a head([2, 18, 9]) = 2
tail [a] -> [a] tail([2, 18, 9]) = [18, 9]
length [a] -> Int length([2, 18, 9]) = 3

Recursive Functions

Length Function

  • Base Case: len([]) = 0
  • Inductive Case: len(x::xs) = 1 + len(xs)

Sum Function

  • Base Case: sum([]) = 0
  • Inductive Case: sum(x::xs) = x + sum(xs)
