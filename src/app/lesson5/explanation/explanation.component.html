Lesson 5: Functional Programming

Example 1: Simple Interest Calculation

Calculate Simple Interest (SI) and Total Amount (Amt) using functional programming. 

          si(p, n, r) = p * n * (r / 100)
          amt(p, n, r) = p + si(p, n, r)
          Example inputs: P = 2000, N = 3, R = 15

SI = 900

Amt = 2900

Example 2: Term-Rewriting Model

Simulating step-by-step term rewriting:

          si = p * n * r / 100
          amt = p + si
          Steps:
          si = 2000 * 3 * 15 / 100
          amt = 2000 + si
          Result:
          SI = 900
          Amt = 2900

Example 3: State-Change Model

Demonstration of errors caused by statement order in imperative programming: 

          # Incorrect order
          si = p * n * r / 100
          p = 2000
          n = 3
          r = 15
          Error: NameError: name 'p' is not defined
  
          # Correct order
          p = 2000
          n = 3
          r = 15
          si = p * n * r / 100
          Output: SI = 900

Example 4: Fibonacci Sequence

Compute the Fibonacci sequence using recursion in a functional programming style: 

            fib(0) = 0
            fib(1) = 1
            fib(n) = fib(n-1) + fib(n-2)
      
            Example:
            fib(5) = fib(4) + fib(3)
            = (fib(3) + fib(2)) + (fib(2) + fib(1))
            = 5

fib(5) = 5

fib(10) = 55

Example 5: Factorial Calculation

Compute the factorial of a number using recursion: 

            factorial(0) = 1
            factorial(n) = n * factorial(n-1)
      
            Example:
            factorial(5) = 5 * 4 * 3 * 2 * 1 = 120

factorial(5) = 120

factorial(7) = 5040

Example 6: Greatest Common Divisor (GCD)

Calculate the GCD of two numbers using recursion: 

            gcd(a, 0) = a
            gcd(a, b) = gcd(b, a % b)
      
            Example:
            gcd(48, 18)
            = gcd(18, 48 % 18)
            = gcd(18, 12)
            = gcd(12, 6)
            = gcd(6, 0)
            = 6

gcd(48, 18) = 6

gcd(101, 103) = 1

Example 7: Power Calculation

Compute the power of a number using recursion: 

            power(base, 0) = 1
            power(base, exp) = base * power(base, exp-1)
      
            Example:
            power(2, 3) = 2 * 2 * 2 = 8

power(2, 3) = 8

power(5, 4) = 625
