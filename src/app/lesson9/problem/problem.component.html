Problem 1:

Define a binary function area that computes the area of a rectangle, given the length and breadth.

Answer: area: Num a => (a, a) -> a
area (len, br) = len * br

Problem 2:

What is the result of area (5, 10) for the function area from Problem 1?

Answer: 50 : Int

Problem 3:

Define a function vol to compute the volume of a cylinder, given its radius and height.

Answer: vol: Num a => (a, a) -> a
vol (rad, ht) = pi * rad^2 * ht

Problem 4:

What is the type of the function vol from Problem 3?

Answer: vol : Num a => (a, a) -> a

Problem 5:

Define area1 for the area of a rectangle, where both sides are integers.

Answer: area1: (Int, Int) -> Int
area1 (len, br) = len * br

Problem 6:

What happens if you call area1 (5.0, 10.0)?

Answer: Error: Type error. area1 expects two Int arguments, not Float.

Problem 7:

Define area2 for the area of a rectangle, where both sides are floats.

Answer: area2: (Float, Float) -> Float
area2 (len, br) = len * br

Problem 8:

What is the result of area2 (5.0, 10.0)?

Answer: 50.0 : Float

Problem 9:

Define the curried version of the function area1 as area1c.

Answer: area1c = curry area1

Problem 10:

What is the result of area1c 10 20?

Answer: 200 : Int

Problem 11:

What is the result of area1c 10?

Answer: Int -> Int

Problem 12:

Define the uncurried version of area1c as area1u.

Answer: area1u = uncurry area1c

Problem 13:

Define power for exponentiation of a base to an integer power.

Answer: power: Num a => a -> Int -> a
power base n = base ^ n

Problem 14:

What is the type of power from Problem 13?

Answer: power : Num a => a -> Int -> a

Problem 15:

What is the result of power 2 3?

Answer: 8 : Int

Problem 16:

What is the result of power 2.0 3?

Answer: 8.0 : Float

Problem 17:

Define mult to compute the multiplication of two integers without using the multiplication operator.

Answer: mult: Int -> Int -> Int
mult m n = if n == 0 then 0 else m + mult m (n - 1)

Problem 18:

What is the result of mult 4 3?

Answer: 12 : Int

Problem 19:

What happens if you call mult 5 0?

Answer: 0 : Int

Problem 20:

Define a ternary function sum3 that computes the sum of three integers.

Answer: sum3: Int -> Int -> Int -> Int
sum3 x y z = x + y + z

Problem 21:

Define a function to compute the factorial of a number recursively.

Answer: factorial: Int -> Int
factorial 0 = 1
factorial n = n * factorial (n - 1)

Problem 22:

What is the result of factorial 5?

Answer: 120 : Int

Problem 23:

Define a function to compute the Fibonacci sequence recursively.

Answer: fibonacci: Int -> Int
fibonacci 0 = 0
fibonacci 1 = 1
fibonacci n = fibonacci (n - 1) + fibonacci (n - 2)

Problem 24:

What is the result of fibonacci 6?

Answer: 8 : Int

Problem 25:

Define a function that computes the greatest common divisor (GCD) of two integers using recursion.

Answer: gcd: Int -> Int -> Int
gcd a 0 = a
gcd a b = gcd b (a mod b)

Problem 26:

What is the result of gcd 56 98?

Answer: 14 : Int

Problem 27:

Define a function that checks if a number is prime.

Answer: isPrime: Int -> Bool
isPrime 2 = True
isPrime n = all (\x -> n `mod` x /= 0) [2..n-1]

Problem 28:

What is the result of isPrime 7?

Answer: True : Bool

Problem 29:

Define a function that calculates the sum of all elements in a list of integers.

Answer: sumList: [Int] -> Int
sumList [] = 0
sumList (x:xs) = x + sumList xs

Problem 30:

What is the result of sumList [1, 2, 3, 4, 5]?

Answer: 15 : Int

Problem 31:

Define a function that calculates the length of a list.

Answer: lengthList: [a] -> Int
lengthList [] = 0
lengthList (x:xs) = 1 + lengthList xs

Problem 32:

What is the result of lengthList [1, 2, 3, 4]?

Answer: 4 : Int

Problem 33:

Define a function that finds the maximum element in a list of integers.

Answer: maxList: [Int] -> Int
maxList [x] = x
maxList (x:xs) = max x (maxList xs)

Problem 34:

What is the result of maxList [1, 3, 2, 8, 5]?

Answer: 8 : Int

Problem 35:

Define a function to reverse a list.

Answer: reverseList: [a] -> [a]
reverseList [] = []
reverseList (x:xs) = reverseList xs ++ [x]

Problem 36:

What is the result of reverseList [1, 2, 3]?

Answer: [3, 2, 1] : [Int]

Problem 37:

Define a function to compute the power set of a list (the set of all subsets).

Answer: powerSet: [a] -> [[a]]
powerSet [] = [[]]
powerSet (x:xs) = let ps = powerSet xs in ps ++ map (x:) ps

Problem 38:

What is the result of powerSet [1, 2]?

Answer: [[], [1], [2], [1, 2]] : [[Int]]

Problem 39:

Define a function to check if a list is sorted in ascending order.

Answer: isSorted: [Int] -> Bool
isSorted [] = True
isSorted [x] = True
isSorted (x:y:xs) = x <= y && isSorted (y:xs)

Problem 40:

What is the result of isSorted [1, 2, 3, 4]?

Answer: True : Bool
