Lesson 6: Proof by Induction and Its Relation to Recursive Functions

Introduction

Mathematical induction is a proof technique used to prove that a statement is true for all natural numbers. It includes the base case, the induction hypothesis, and the induction step.

Induction is closely related to recursion in programming, where solutions to problems depend on solutions to smaller subproblems.

Example: Sum of the First n Natural Numbers

Prove that the sum ( 0 + 1 + 2 + ldots + n = fracn(n+1)2 ).

  1. Base Case: ( P(0) ) holds since both sides evaluate to 0.
  2. Induction Hypothesis: Assume ( P(k): 0 + 1 + 2 + ldots + k = frack(k+1)2 ) is true for ( k geq 0 ).
  3. Induction Step: Show ( P(k+1) ): ( frack(k+1)2 + (k+1) = frac(k+1)(k+2)2 ).

By induction, ( P(n) ) is true for all ( n geq 0 ).

Recursive Functions

Sum of Natural Numbers

  
  sumupto1(0) = 0
  sumupto1(n) = sumupto1(n-1) + n

Factorial Function

  
  factorial(0) = 1
  factorial(n) = n * factorial(n-1)

Key Takeaways

  • Induction proves propositions for all natural numbers.
  • Recursion solves problems by breaking them into smaller subproblems.
  • Structural induction extends induction to recursively defined data structures.
