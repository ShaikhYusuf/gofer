Example 1: Sum of Natural Numbers

Prove: The sum 1 + 2 + ... + n = n(n+1)/2.

Solution:

  1. Base Case: For n = 1, 1 = 1(1+1)/2, which holds true.
  2. Induction Hypothesis: Assume 1 + 2 + ... + k = k(k+1)/2 is true for some k ≥ 1.
  3. Induction Step:

    Prove 1 + 2 + ... + k + (k+1) = (k+1)(k+2)/2.

                LHS = k(k+1)/2 + (k+1)
                = (k+1)(k+2)/2

Conclusion: The formula holds for all n ≥ 1.

Example 2: Sum of Squares

Prove: The sum 1² + 2² + ... + n² = n(n+1)(2n+1)/6.

Solution:

  1. Base Case: For n = 1, 1² = 1(1+1)(2×1+1)/6, which holds true.
  2. Induction Hypothesis: Assume 1² + 2² + ... + k² = k(k+1)(2k+1)/6.
  3. Induction Step:

    Prove 1² + 2² + ... + k² + (k+1)² = (k+1)(k+2)(2(k+1)+1)/6.

                LHS = k(k+1)(2k+1)/6 + (k+1)²
                = (k+1)(k(2k+1)/6 + (k+1))
                = (k+1)(k+2)(2k+3)/6

Conclusion: The formula holds for all n ≥ 1.

Example 3: Factorial

Prove: n! = n × (n-1)! for all n ≥ 1.

Solution:

  1. Base Case: For n = 1, 1! = 1, which holds true.
  2. Induction Hypothesis: Assume k! = k × (k-1)! is true.
  3. Induction Step:

    Prove (k+1)! = (k+1) × k!.

                LHS = (k+1) × k!
                = (k+1)!

Conclusion: The formula holds for all n ≥ 1.

Example 4: Fibonacci Sequence

Prove: F(n) = F(n-1) + F(n-2) for n ≥ 2, where F(0) = 0 and F(1) = 1.

Solution:

  1. Base Case: For n = 2, F(2) = F(1) + F(0) = 1 + 0 = 1.
  2. Induction Hypothesis: Assume F(k) = F(k-1) + F(k-2).
  3. Induction Step:

    Prove F(k+1) = F(k) + F(k-1).

                LHS = F(k) + F(k-1)
                = F(k+1)

Conclusion: The recurrence holds for all n ≥ 2.

Example 5: Exponentiation

Prove: 2^n = 2 × 2^n-1 for all n ≥ 0.

Solution:

  1. Base Case: For n = 0, 2^0 = 1, which matches the formula.
  2. Induction Hypothesis: Assume 2^k = 2 × 2^k-1.
  3. Induction Step:

    Prove 2^(k+1) = 2 × 2^k.

                LHS = 2 × 2^k
                = 2^(k+1)

Conclusion: The formula holds for all n ≥ 0.

Example 6: Geometric Series

Prove: The sum 1 + r + r² + ... + r^n = (1 - r^(n+1)) / (1 - r) for r ≠ 1.

Solution:

  1. Base Case: For n = 0, 1 = (1 - r^(0+1)) / (1 - r) = 1, which is true.
  2. Induction Hypothesis: Assume 1 + r + r² + ... + r^k = (1 - r^(k+1)) / (1 - r) holds for some k ≥ 0.
  3. Induction Step:

    Prove 1 + r + ... + r^k + r^(k+1) = (1 - r^(k+2)) / (1 - r).

              LHS = (1 - r^(k+1)) / (1 - r) + r^(k+1)
              = (1 - r^(k+1) + r^(k+1)(1 - r)) / (1 - r)
              = (1 - r^(k+2)) / (1 - r)

Conclusion: The formula holds for all n ≥ 0.

Example 7: Even Numbers

Prove: The sum of the first n even numbers is 2 + 4 + ... + 2n = n(n+1).

Solution:

  1. Base Case: For n = 1, 2 = 1(1+1) = 2, which holds true.
  2. Induction Hypothesis: Assume 2 + 4 + ... + 2k = k(k+1) holds for some k ≥ 1.
  3. Induction Step:

    Prove 2 + 4 + ... + 2k + 2(k+1) = (k+1)((k+1)+1).

              LHS = k(k+1) + 2(k+1)
              = (k+1)(k+2)

Conclusion: The formula holds for all n ≥ 1.

Example 8: Divisibility

Prove: 3 | (4^n - 1) for all n ≥ 1.

Solution:

  1. Base Case: For n = 1, 4^1 - 1 = 3, which is divisible by 3.
  2. Induction Hypothesis: Assume 3 | (4^k - 1) holds for some k ≥ 1.
  3. Induction Step:

    Prove 3 | (4^(k+1) - 1).

              LHS = 4^(k+1) - 1
              = 4 × 4^k - 1
              = (4 × (4^k - 1)) + (4 - 1)
              = divisible by 3

Conclusion: 3 | (4^n - 1) for all n ≥ 1.

Example 9: Inequality

Prove: 2^n > n² for all n ≥ 5.

Solution:

  1. Base Case: For n = 5, 2^5 = 32 > 25 = 5², which holds true.
  2. Induction Hypothesis: Assume 2^k > k² for some k ≥ 5.
  3. Induction Step:

    Prove 2^(k+1) > (k+1)².

              LHS = 2 × 2^k > 2 × k²
              > (k+1)² for k ≥ 5

Conclusion: 2^n > n² for all n ≥ 5.
