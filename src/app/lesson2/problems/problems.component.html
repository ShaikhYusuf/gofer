Create a tuple containing an integer, a character, and a Boolean.

-- Result: (42, 'X', True) : (Int, Char, Bool)

Problem 2

-- Result: Error, since fst works on pairs, not triples. Use pattern matching instead:

Check if tuples have a fixed length.
