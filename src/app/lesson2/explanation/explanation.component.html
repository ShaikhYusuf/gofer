Lesson 2: Compound Data Types in Gofer

What is a Tuple?

A tuple is a collection of values enclosed in parentheses (). It can store multiple data types but has a fixed size.

  • Example: (4, 3.14, 'Z', True)
  • Type: (Int, Float, Char, Bool)

Tuples can also be nested. For example: ((1, 2), (3.0, 'A')) has the type ((Int, Int), (Float, Char)).

What is a List?

A list is a sequence of values enclosed in square brackets []. All elements must be of the same type, but lists can grow or shrink.

  • Example: [1, 2, 3]
  • Type: [Int]

Lists can also represent strings, where a string is a list of characters. For example: "hello" == ['h', 'e', 'l', 'l', 'o'].

Comparing Tuples and Lists

Feature Tuples Lists
Symbols Parentheses () Square Brackets []
Types Can be mixed (heterogeneous) Must be the same (homogeneous)
Size Fixed Flexible

Operations on Lists

  • Concatenation: Use ++ to join lists.
    • [1, 2] ++ [3, 4] → [1, 2, 3, 4]
    • "abc" ++ "def" → "abcdef"
  • Ranges: Quickly create sequences of numbers.
    • [1 ... 5] → [1, 2, 3, 4, 5]
    • [5, 4 ... 1] → [5, 4, 3, 2, 1]
