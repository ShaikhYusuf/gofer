Problem 35

Write a recursive function to calculate the factorial of a number n.

Solution:

factorial 0 = 1
factorial n = n * factorial (n - 1)

Problem 36

Write a function that checks if a number is even.

Solution:

isEven n = n `mod` 2 == 0

Problem 37

Write a function to sum the elements of a list of numbers.

Solution:

sumList [] = 0
sumList (x:xs) = x + sumList xs

Problem 38

Define a function to find the maximum element in a list.

Solution:

maxList [x] = x
maxList (x:xs) = max x (maxList xs)

Problem 39

Write a function to reverse a list.

Solution:

reverseList [] = []
reverseList (x:xs) = reverseList xs ++ [x]

Problem 40

Define a function that checks if a number is prime.

Solution:

isPrime 1 = False
isPrime n = all (\x -> n `mod` x /= 0) [2..n-1]

Problem 41

Use pattern matching to extract the first and second elements of a tuple.

Solution:

let (x, y) = (10, 20) in (x, y)

Problem 42

Use a list to represent a sequence of Fibonacci numbers.

Solution:

fibonacci 0 = 0
fibonacci 1 = 1
fibonacci n = fibonacci (n - 1) + fibonacci (n - 2)

Problem 43

Write a function to filter all even numbers from a list.

Solution:

filterEven [] = []
filterEven (x:xs) = if even x then x : filterEven xs else filterEven xs

Problem 44

Write a function to count the number of occurrences of an element in a list.

Solution:

countOccurrences _ [] = 0
countOccurrences x (y:ys) = (if x == y then 1 else 0) + countOccurrences x ys

Problem 45

Define a function to check if a string is a palindrome.

Solution:

isPalindrome str = str == reverse str

Problem 46

Write a function to return the nth element of a list.

Solution:

nthElement 0 (x:xs) = x
nthElement n (x:xs) = nthElement (n - 1) xs

Problem 47

Define a function to apply a function to each element of a list.

Solution:

mapFunction _ [] = []
mapFunction f (x:xs) = f x : mapFunction f xs

Problem 48

Write a function to calculate the length of a list.

Solution:

lengthList [] = 0
lengthList (_:xs) = 1 + lengthList xs

Problem 49

Write a function that returns the head of a list.

Solution:

headOfList (x:xs) = x

Problem 50

Write a function to merge two sorted lists into a single sorted list.

Solution:

merge [] ys = ys
merge xs [] = xs
merge (x:xs) (y:ys) = if x < y then x : merge xs (y:ys) else y : merge (x:xs) ys
