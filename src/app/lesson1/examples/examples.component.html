Gofer Component Examples

1. Simple Expressions

        ? 2
        2 : Int
  
        ? 2 + 2
        4 : Int
  
        ? 2 * 3
        6 : Int
  
        ? 2 + 3.5
        ERROR: Type error in application
        *** expression : 2 + 3.5
        *** term : 3.5
        *** type : Float
        *** does not match : Int

Explanation: Integer operations like 2 + 2 result in an integer, but adding an integer and a float, like 2 + 3.5, causes a type error.

2. Boolean Expressions

        ? True
        True : Bool
  
        ? False
        False : Bool
  
        ? 2 > 1
        True : Bool
  
        ? 3 < 2
        False : Bool

Explanation: True and False are boolean values. Expressions like 2 > 1 evaluate to True.

3. Character Expressions

        ? 'a'
        'a' : Char
  
        ? '2'
        '2' : Char
  
        ? ' '
        ' ' : Char

Explanation: Characters such as 'a', '2', and ' ' are of type Char.

4. Expressions with Exponentiation

        ? 2^5
        32 : Int
  
        ? 2.0^5
        32.0 : Float
  
        ? 2^5.0
        ERROR: Type error in application
        *** expression : 2^5.0
        *** term : 5.0
        *** type : Float
        *** does not match : Int

Explanation: The operation 2^5 results in an integer, while 2.0^5 results in a float. Type errors occur when the base and exponent are mismatched in types.

5. Equality and Inequality Expressions

        ? 2 + 2 == 4
        True : Bool
  
        ? 2 + 2 == 5
        False : Bool
  
        ? 2 /= 3
        True : Bool
  
        ? 2 /= 4
        False : Bool
  
        ? True == False
        False : Bool
  
        ? True /= False
        True : Bool

Explanation: Simple equality and inequality checks are demonstrated with operations like 2 + 2 == 4 and 2 /= 3.

6. String Expressions

        ? "hello, world!"
        "hello, world!" : String
  
        ? :t "hello"
        "hello" : String

Explanation: Strings like "hello, world!" are of type String, and the type is shown with the :t operator.
