Data and Data Types

Simple Data Types

  • Int: Integer values (e.g., 2, -5)
  • Float: Decimal numbers (e.g., 2.1, 1.23e+06)
  • Char: Single characters (e.g., 'a', '2')
  • Bool: Boolean values (True, False)

Examples

Try the following Gofer expressions in the interpreter:

  • 5 (Integer)
  • 3.14 (Float)
  • 'g' (Character)
  • True (Boolean)

Operators and Expressions

Arithmetic Operators

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • ^: Exponentiation

Boolean Operators

  • &&: Logical AND
  • ||: Logical OR

Examples

Here are some examples you can test in Gofer:

  • 2 + 3 = 5 (Addition)
  • 8 - 3 = 5 (Subtraction)
  • 4 * 5 = 20 (Multiplication)
  • 10 / 2 = 5 (Division)
  • 2 ^ 3 = 8 (Exponentiation)
  • True && False = False (Logical AND)
  • True || False = True (Logical OR)

Recursion and Higher-Order Functions

Recursion Example

Recursion allows a function to call itself. This is a key concept in functional programming.

  • Factorial Function:
    • factorial n = if n == 0 then 1 else n * factorial (n - 1)

    Test the function by calling factorial 5 which will return 120.

Higher-Order Function Example

Higher-order functions are functions that take other functions as arguments or return them as results.

  • Map Function:
    • map f lst = [f x | x <- lst]

    This function applies the function f to each element of the list lst.

    Test the function by calling map (+1) [1, 2, 3], which will return [2, 3, 4].
