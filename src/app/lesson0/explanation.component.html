IPMF Lesson 0: Installation and Testing of the Gofer Programming Platform

What is Gofer, and Why Do We Use It?

Gofer is a programming language used for learning programming and mathematics. It’s a dialect of Haskell, which focuses on functional programming.

Functional programming is a way to describe what to do instead of writing step-by-step instructions. Gofer uses immutable values, meaning once a value is assigned, it cannot change.

Gofer's Unique Features

  • No Variables: Gofer doesn’t use traditional variables. Instead, it uses = for definitions, making all values unchangeable.
  • Good for Equational Reasoning: Gofer simplifies math-like problem-solving.

How Does Gofer Work?

Think of Gofer as a super-smart calculator. It solves math problems and runs scripts to handle complex tasks. The main components are:

  • The Prelude: Predefined tools and functions that are always available.
  • Your Scripts: Custom programs written by you for specific tasks.

Steps to Install and Run Gofer

Step 1: Download the Platform

Get the Gofer platform from a zip file or a shared folder. Extract or download the files to a convenient location like the desktop.

Step 2: Running Gofer

For Windows, double-click gofer32.exe to start the interpreter.

For Linux, use the terminal to navigate to the Gofer folder and execute the command ./gofer.

Testing Gofer

Open the Gofer interpreter and type 2 + 2. If it responds with 4:Int, your installation is successful!

Working with Scripts

A script is a program you create to solve specific problems. Save scripts with the extension .gs and load them into the interpreter using the command:

:load myprogram.gs

To create scripts:

  • Use Notepad or Notepad++ on Windows.
  • Use gedit or another text editor on Linux.

Improving Visibility (Windows Only)

Change the font in the Command-Line Interface by right-clicking its border, selecting Properties, and adjusting the font size under the Font tab.
