Programming Concepts

  1. Function Composition
    Problem: Given ( f(x) = x^2 ) and ( g(x) = x + 3 ), find ( (f circ g)(2) ).
    Solution: ( g(2) = 5, f(5) = 25. ) So, ( (f circ g)(2) = 25 ).
  2. Higher-Order Function
    Problem: ( applyTwice(f, x) = f(f(x)) ). Find ( applyTwice(f = (x + 3), x = 2) ).
    Solution: ( f(2) = 5, f(5) = 8 ). So, ( applyTwice = 8 ).
  3. Curried Function
    Problem: ( multiply(x) = (y to x times y) ). Find ( multiply(3)(4) ).
    Solution: ( 3 times 4 = 12 ).
  4. Lambda Function
    Problem: Define ( f(x) = lambda x : x + 5 ). Find ( f(7) ).
    Solution: ( f(7) = 7 + 5 = 12 ).
  5. Map Function
    Problem: ( map(f, [1, 2, 3]) ) where ( f(x) = x^2 ).
    Solution: Output: [1, 4, 9].
