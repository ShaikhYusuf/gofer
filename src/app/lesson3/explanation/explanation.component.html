Introduction to Functional Programming

Lesson 3: Functions, Types, and Application in Gofer

What is a Function?

A function is a mapping from a domain to a range, ensuring each input in the domain has a unique corresponding output in the range.
Example: square(x) = x^2

Function Types

In Gofer, function types are written as a -> b, where:

  • a: The input type (domain)
  • b: The output type (range)

Example: cos : Float -> Float maps a Float to another Float.

Function Application

In Gofer, function application is explicit using the dot operator (.).

Example: cos . pi applies the cos function to pi.

Functions can also be composed using the semicolon (;) operator:
(sq ; dbl ; succ) . 10 computes the result of applying the functions in sequence.

Built-In Functions

Gofer provides built-in functions for tuples and lists:

  • fst, snd: Access tuple components
  • head, tail: Access list components

Example: head . [0 ... 5] returns 0.
